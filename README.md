# ml-embeddings
The following repo provides a framework to generate ML embeddings for unstructured data on the [lakehouse](https://www.databricks.com/product/data-lakehouse).

This pipeline takes advantage of available open-source technologies such as:

```
Delta Lake
Towhee
Milvus
```

A link to the associated blog post can be found [here](https://amirlyall.medium.com/simple-and-scalable-solutions-for-unstructured-data-on-the-lakehouse-1ee1f240cdf9).
